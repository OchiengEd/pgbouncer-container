# PG Bouncer
This repository intends to build a pgbouncer image that is based on Red Hat UBI images for use with certified operators.

## Environment Variables

- `DB_USER`

- `DB_PASSWORD`

- `DB_PORT`

- `DB_HOST`

- `AUTH_TYPE`

- `DEFAULT_POOL_SIZE`

- `IGNORE_STARTUP_PARAMETERS`

- `QUERY_TIMEOUT`

- `RESERVE_POOL_TIMEOUT`

- `RESERVE_POOL_SIZE`

- `POOL_MODE`

- `DEFAULT_POOL_SIZE`

- `LISTEN_ADDR`

- `MAX_CLIENT_CONN`

- `MAX_DB_CONNECTIONS`

- `MIN_POOL_SIZE`
